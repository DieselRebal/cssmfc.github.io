    Editing Tips

    The above demo is using % values which makes the layout semi-responsive.

    When adding your graphics (images) use the in-line CSS to style them up and you can use the same % value in order to make them responsive

    Change the top Model's image, replace only the image URL path

    <img src="images/pic12.jpg" style="display:block;width:100%;height:auto;margin:0 auto;"/>
