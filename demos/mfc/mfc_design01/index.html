Install Instructions

Download all files and start customization
DO NOT FORGET to upload your images online and replace the default direct url path for provided graphics by editing the source markup.
Do NOT skip this step because you'll have a "broken" template with no graphic elements if you install it on MyFreeCams.

  1. Open the source code provided using your favorite markup editor (Notepad++ for example) or use the Source Editor tool to design your template offline
  2. Edit, replace and add your own content
  3. Install the edited code on your profile page -- paste the code in About Me box
